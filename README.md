# Nakhon Sawan Smart City Analytics

Premium Next.js dashboard for analyzing smart city projects, budget allocation, executive insights, reporting, and media review.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Highlights

- Smart filters with instant recalculation
- Executive summary and strategic insights
- Beautiful custom charts and glassmorphism UI
- CSV / summary / deep-analysis report downloads
- Media center powered by local `Source` folder
